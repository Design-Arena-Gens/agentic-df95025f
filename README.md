# Instagram Analyzer

Web tool to analyze public Instagram account metrics (best-effort) and compare multiple handles side-by-side. Built with Next.js and deployable to Vercel.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Notes
- Uses public profile page metadata. Some metrics may be unavailable if Instagram blocks automated requests.
- Export results to CSV from the UI.
